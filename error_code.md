# HTTP状态码
|状态码   |含义   |说明   |
| ------------ | ------------ | ------------ |
|  200 | OK  | 请求成功  |
| 201  |  CREATED |  创建成功 |
| 202  | ACCEPTED  |  更新成功 |
|  204 | NO CONTENT  | 删除成功  |
| 301  | MOVED PERMANENTLY  | 永久重定向  |
|  400 | BAD REQUEST  |  请求包含不支持的参数 |
| 401  | UNAUTHORIZED  | 未授权  |
| 403  | FORBIDDEN  |  被禁止访问 |
|  404 | NOT FOUND  | 请求的资源不存在  |
|  429 | TOO MANY REQUESTS  | 请求数过多  |
|  500 | INTERNAL SERVER ERROR  | 服务器错误  |

# 错误码说明
#### ```100x```说明如下
|  返回码 | 说明  |
| ------------ | ------------ |
| 0  | 成功  |
| 1000  | 请求参数错误  |
| 1001  | 资源未找到  |
| 1002  | 认证失败  |
| 1003  | 权限不足，禁止访问  |
| 999  | 未知错误  |


#### ```200x```说明如下
|  返回码 | 说明  |
| ------------ | ------------ |
| 2001  | 验证码错误  |
| 2002  | 已经申请了appkey，不能重复申请  |
| 2003  | appkey错误  |
